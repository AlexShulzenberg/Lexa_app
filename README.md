# Lexa - Приложение-помощник в изучении французского
Веб-приложение для изучения французских слов с упрощенной системой 
интервального повторения и AI-функциями для перевода.

# О проекте

**Проблема:** изучающие французский язык не имеют удобного инструмента
для систематической работы с новыми словами из книг и статей.

**Решение:** Lexa позволяет сканировать текст с изображений, автоматически
извлекать слова, переводить их через AI и изучать через уроки.

# Функциональность

- Коллекции — организация слов по темам
- Уроки — 4 типа упражнений (перевод FR→RU, RU→FR, заполнение пропуска, задача на соответствие слова и перевода)
- OCR — распознавание текста с фотографий через OCR.space API
- AI — автоперевод и анализ текста через Google Gemini API
- Прогресс — графики статистики, страйки, уровни владения словами
- Импорт/Экспорт — загрузка коллекций через JSON

# Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/AlexShulzenberg/Lexa_app
cd lexa-app
```

### 2. Создать виртуальное окружение

```bash
python -m venv .venv
```

Активировать:

- **Windows:**
```bash
.venv\Scripts\activate
```

- **macOS / Linux:**
```bash
source .venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить переменные окружения

Создай файл `.env` в корне проекта:

```env
SECRET_KEY=***
GEMINI_API_KEY=***
OCR_SPACE_API_KEY=***
DEBUG=True
```

Где получить ключи бесплатно:
- **Gemini API** — https://aistudio.google.com/
- **OCR.space API** — https://ocr.space/ocrapi/freekey

### 5. Применить миграции

```bash
python manage.py migrate
```

### 6. Запустить сервер

```bash
python manage.py runserver
```

Открыть в браузере: http://127.0.0.1:8000
